OpenFOAM Tool
A OpenFOAM Toolbox

Overview

This software toolbox was created for OpenFOAM v6.
This software includes common OpenFOAM functions, such as blockMesh, snappyHexMesh and simpleFoam and pisoFoam. There are also some custom features not directly included from OpenFOAM, such as a residual plot, and a modelViewer.

How to build

1.Qt Framework

First you need to install Qt 5.12.11 from here and follow the instructions. Do not forget to install the QT Charts Addon from the installer.
Do not forget to add the /<qt-installation>/5.12.11/gcc_64/bin to the $PATH variable.

2.Assimp

Secondly you need to install the Open Asset Import Library (assimp) from here and follow the instructions listed there.
Do not forget to copy the library files in assimp/bin to usr/lib/

3.Build

If you have installed Qt and Assimp correctly you should now be able to execute the following commands.

  1. mkdir build
  2. cd build
  3. qmake ../source/openfoamtool.pro
  4. make
  5. make clean

4.Finishing touches

You may need to add "export MESA_GL_VERSION_OVERRIDE=3.3" to your ~/.bashrc if you encounter an openGL version error.
